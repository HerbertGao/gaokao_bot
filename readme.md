## Introduction
高考倒计时 @gaokao_bot 是一个 Telegram Bot，可以实现通过发送 Message、Inline Query 方式获取当前时间的高考倒计时。

Bot 同时集成了 XXL-JOB 框架，可实现定时向指定会话发送倒计时。
